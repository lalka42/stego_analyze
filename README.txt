Данная программа предназначена для анализа дампов сетевого трафика (.pcap, /.pcapng)
на наличие признаков сетевой стеганографии.
Данная программа использует Wireshark для парсинга дампов, поэтому без Wireshark программа работать не будет!!!
Данная программа использует библиотеку Scikit Sklearn.
Баги есть, потому что программа в активной разработке.

!Путь к папке с программой должен быть без пробелов
!Путь к папке с дампом тоже должен быть без пробелов
p.s. Возможно уже всё-равно на пробелы, ещё не тестил
